1. Чтение файлов данных и их интерпретация
   Заполнение словаря игроков и словаря сохранений

2. Запрос имени игрока
	Приветствие
	Вывод раздела помощи

3. Если имя игрока отсутствует в словаре игроков:
	добавление нового игрока с нулевой статистикой

# Суперцикл - Главное меню

	3.1 Ожидание ввода команды
	
	3.2 начать/н/start

		3.2.1 Запрос режима игры
		
		3.2.2 Одиночная игра:
			
			3.2.2.1 Выбор уровня сложности бота

			3.2.2.2 Добавление имени бота ('easy bot'/'hard bot') в список активных игроков
			
			3.2.2.3 Добавление функции хода бота в список функций ходов

		3.2.3 Кооперативная игра:
			
			3.2.3.1 Запрос имени игрока
		
			3.2.3.2 Если имя игрока отсутствует в словаре игроков:
						Добавление нового игрока в словарь игроков

			3.2.3.3 Добавление имени второго игрока в список активных игроков

			3.2.3.4 Добавление функции запроса хода игрока в список функций ходов
		
		3.2.4 Проверка сохраненных игр
				
			Если есть:
				Запрос загрузки сохраненной игры
					
				Если да:
					Заполнение параметров сохраненной партии
				Если нет:
					-> 3.2.5

			

		3.2.5 Запрос токена для 1-го игрока
		
		3.2.6 Если игрок выбирает "О", записать в параметр "Х" второго игрока
		
		3.2.7 Запрос размера игрового поля
			Добавление параметров игрового поля
			Добавление возможных выйгрышных комбинации для соответствующего размера игрового поля
				
		3.2.8 Вывод пустого игровово поля
		
		# цикл по количеству клеток игрового поля (1 итерация - один ход)

			3.2.8.1 Запрос хода игрока 
		
				3.2.8.1.1 Проверка хода на допустимый диапозон значений (Перезапрашивать ход пока не будет введенно допустимое значение)

				3.2.8.1.2 Проверить занята клетка или нет (Перезапрашивать ход пока не будет выбрана пустая клетка)

			3.2.8.2 Или вычисление хода бота

			3.2.8.3 Добавить ход в список сделанных ходов

			3.2.8.4 Проверка всех ходов на выйгрышные комбинации 
					Если выйгрышная комбинация входит в список сделанных ходов:
						- Прервать цикл
						- обновить статистики и файл с данными игроков (победа/поражение соответственно)
						- Вывести имя победителя
						
			3.2.8.5 Проверка всех ходов на возможность продолжения партии
					Если возможных выйгрышных комбинаций больше нет, предложить закончить партию в ничью
					Если игроки согласны закончить партию:
						- Прервать цикл
						- Обновить статистику и файл с данными игроков (ничья)

					Если игроки хотят продолжить:
						- Продолжить цикл до его завершения (пока все ячейки не будут заняты)
						- Обновить статистику и файл с данными игроков (ничья)		

			3.2.8.6 Вывод игрового поля со сделанными ходами

		3.2.9 Возврат списка активных игроков и списка функци ходов в первоначальное состояние

		3.2.10 Запрос повторения партии
			Если да:
				Переход к пункту 3.2.1 (Начало новой партии с выбором параметров)
			Если нет:
				Переход к шагу 3.1 (Переход к главному меню)
		
		

4 Ввод команд во время всего игрового процесса 
	- инфо/info (загрузка информации по командам в игре)
	- сохранить/save (Сохранение текущей партии)
		Добавление имен игроков и парметры игры в файл с сохраненными играми
		Завершение цикла игры
		Вывод в главное меню
	- конец/end (Запрос сохрания игры если вызов произведен во время партии, Вывод сообщения о завершении программы)
